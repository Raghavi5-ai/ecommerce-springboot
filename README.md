# Ecommerce Spring Boot Application

## Description
Simple ecommerce backend application built using Spring Boot.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL

## Features
- CRUD operations for users and products
- Order placement with stock validation
- Transaction management
- Exception handling
- Pagination and sorting

## How to Run
1. Update DB details in application.properties
2. Run as Spring Boot Application
